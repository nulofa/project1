Input File Preparation :

File Format
This web accepts query files in .txt(tab-separated) and .tsv format

File Description
Input file must contain gene expression data as 2 columns matrix. The first column must contain the corresponding gene symbols. The first row is sample labels.

A sample query file can be viewed here.

Genes Details
Search is built on a set of 38312 genes. The list of genes used can be found here.

Users do not have to bother about maintaining the right set of genes in the query file. The query preprocessing step handles the user query intelligently to align the genes in a set order along with that it also handles cases when the number of genes is more or less.
